# Actividad: Funciones+Archivos
## Descripción
Tres programas los cuales dan ejemplos del como funcionan los diferentes temas:
-  Struct: En este programa se utilizan distintas estructuras agregar datos en la misma y luego extraerlos.
-  Archivos: Este programa une los archivos con arreglos para ver y practicar el funcionamiento entre ambos.
-  Funciones_Arreglos: Con este programa utilizan distintas funciones junto con arreglos para ver el como se conectan.
## Temas aplicados
- arreglos
- archivos
- funciones
- struct
## Compilación
gcc main.c -o programa
## Ejecución
./programa
