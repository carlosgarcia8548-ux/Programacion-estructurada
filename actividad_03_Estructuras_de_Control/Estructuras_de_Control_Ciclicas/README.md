# Actividad: Estructuras de control ciclicas
## Descripción
Programa en el cual se utilizan distintas estructuras de control(do-while, while, if-else) de manera ciclica para la realizacion de menus con los cuales se haga todo más versatil para el usuario y gracias a estas  estructuras podemos obtener datos como el promedio  de manera repetida. Ademas de notas para entender como funciona el ciclo de vida de un software.
## Temas aplicados
- Estructuras de control
- Ciclo de vida de un software
## Compilación
gcc main.c -o programa
## Ejecución
./programa
