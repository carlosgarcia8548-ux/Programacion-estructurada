# Actividad: Estructuras de control
## Descripción
Programa en el cual se utilizan distintas estructuras de control(do-while, while, if-else), en la cual se realizo un menu, el conseguir un promedio, etc. Esto en ambas actividades realizadas ademas de ver el como funciona un ciclo de vida de software y su funcionamiento.
## Temas aplicados
- Estructuras de control
- Ciclo de vida de un software
## Compilación
gcc main.c -o programa
## Ejecución
./programa
