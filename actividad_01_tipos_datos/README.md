# Actividad: Tipos de datos
## Descripción
Actividad enfocada en el entendimiento de los datos y sus funcionamientos con un programa para ver su dirección de memoria de manera que podamos ver sus bytes.
## Temas aplicados
- tipos de datos
## Compilación
gcc main.c-o programa
## Ejecución
Identificación:A = 22, dirección del entero: 140734498161196  
Identificación:B = true, dirección de booleano: 140734498161195  
Identificación:C = 0.16, dirección de flotante: 140734498161188  
Identificación:D = 2.718281828, dirección de doble: 140734498161176  
Identificación:E = 'A', dirección de caracter: 140734498161188  
Identificación:F = NULL dirección de void*: 0
