# Actividad: Participacion-String
## Descripción
El programa muestra distintas funciones de <string.h> las cuales se utilizan para hacer ejemplos y comprender cual es el objetivo de cada funcion
## Temas aplicados
- <string.h>
## Compilación
gcc main.c -o programa
